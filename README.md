# Optical Flow Motion Analysis

Домашняя работа по Computer Vision.

В проекте реализовано сравнение двух методов оптического потока:

- Lucas–Kanade
- Farnebäck

## Что сделано

Выбран **вариант B** — учебная система анализа движения.

Реализованы:

- поиск и трекинг ключевых точек с помощью Lucas–Kanade
- визуализация траекторий движения
- вычисление плотного оптического потока Farnebäck
- построение карт движения
- получение бинарных масок движущихся областей
- сохранение сравнительных визуализаций и итогового видео
- краткий разбор различий методов и ошибок

## Наблюдения

Lucas–Kanade удобен для отслеживания отдельных точек и траекторий, но хуже работает при blur, перекрытиях и слабой текстуре.

Farnebäck даёт плотное поле движения и лучше подходит для выделения движущихся областей, но может давать шум на тенях, слабом контрасте и границах объектов.

## Вывод

Lucas–Kanade лучше использовать для трекинга признаков и анализа траекторий.
Farnebäck удобнее, когда нужно оценивать движение во всём кадре и получать маски движения.
